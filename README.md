# Simple-Projs_electron
Notes and small projects in ElectronJS

## Requisitos
- Node.js v16+
- NPM ou Yarn

## Como executar
1. Clone este repositório:
   ```bash
   git clone https://github.com/KamiNoKod0mo/Simple-Projs_electron
   cd Simple-Projs_electron
   npm install
   ```

## Execução
```bash
  npm start
```
